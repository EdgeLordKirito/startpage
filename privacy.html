In dieser Datenschutzerklärung informieren wir Sie über die wichtigsten Aspekte der Datenverarbeitung im Rahmen unserer Website.

Kontakt mit uns

Auf dieser Website gibt es keine Möglichkeit der direkten Kontaktaufnahme über Formulare oder E-Mail. Es werden keine personenbezogenen Daten über Formulare oder andere Eingabefelder gespeichert oder verarbeitet.

Hosting

Diese Website wird über GitHub Pages gehostet. Beim Aufrufen der Website werden Verbindungsdaten (z.B. IP-Adresse, Browsertyp) technisch durch GitHub verarbeitet. Weitere Informationen finden Sie in der Datenschutzerklärung von GitHub.

Suchfunktion

Unsere Website bietet eine Suchfunktion, mit der externe Inhalte durchsucht werden können. Die Eingaben in der Suchleiste werden ausschließlich lokal im Browser verarbeitet und nicht an den Betreiber dieser Website übermittelt oder gespeichert.

Externe Links

Unsere Website enthält Links zu externen Webseiten Dritter. Für die Inhalte dieser verlinkten Webseiten wird keine Verantwortung übernommen. Beim Anklicken eines Links verlassen Sie unsere Website. Es findet keine Speicherung oder Analyse der Linknutzung durch uns statt.

Wetteranzeige

Zur Anzeige aktueller Wetterdaten wird auf dieser Website die OpenWeatherMap API genutzt (openweathermap.org). Für die Bestimmung des Standortes basierend auf der IP-Adresse wird der Dienst freeipapi.com verwendet (freeipapi.com).

Hierbei wird Ihre aktuelle IP-Adresse an die genannten Drittanbieter übermittelt, um den ungefähren Standort zu bestimmen und passende Wetterinformationen bereitzustellen. Diese Daten werden von uns nicht gespeichert oder weiterverarbeitet. Bitte beachten Sie die jeweiligen Datenschutzerklärungen der genannten Dienste:

  • Datenschutzerklärung OpenWeatherMap
  • Datenschutzerklärung FreeIPAPI

Local Storage

Zur Verbesserung der Benutzerfreundlichkeit speichern wir bestimmte Einstellungen lokal in Ihrem Browser (Local Storage). Dazu zählen die gewählte Suchmaschine, das bevorzugte Farbschema sowie die zuletzt angezeigten Wetterdaten. Diese Informationen werden ausschließlich lokal auf Ihrem Gerät gespeichert und nicht an uns oder Dritte übermittelt.
